一起俯卧撑
=================

![Image text](http://github.com/MSstudent/-Project-Health/raw/master/sample.jpg)

#项目简介

* 一个诱惑你做俯卧撑的小软件，把你的女友or女神照片放上来，然后尽情kiss吧……

---

#使用方法

* 双击计数器可开始计数，亲到图片计数加1，再次双击计数器计数结束。
* 结束后可分享到微博，预留分享人人，但未添加。
* 图表可统计最近7日每日俯卧撑个数。
* 未开始计数时单击图片可更换图片。
